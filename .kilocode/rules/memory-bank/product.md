# Prize Raffle - Опис продукту

## Що це за продукт?
Prize Raffle - це веб-додаток для проведення справедливих розіграшів призів з візуальними ефектами та математичним обґрунтуванням чесності. Додаток створений для організаторів заходів, які потребують прозорого та справедливого способу розподілу призів серед учасників.

## Проблеми, які вирішує

### 1. Недовіра до справедливості розіграшів
**Проблема:** Учасники часто сумніваються в чесності традиційних методів розіграшу.
**Рішення:** Використання криптографічно стійкого генератора випадкових чисел та вбудованих статистичних тестів для математичного підтвердження справедливості.

### 2. Складність організації розіграшів
**Проблема:** Ручне проведення розіграшів займає багато часу та схильне до помилок.
**Рішення:** Автоматизований процес з інтуїтивним інтерфейсом, який дозволяє швидко налаштувати та провести розіграш.

### 3. Втрата даних та результатів
**Проблема:** Ризик втрати інформації про учасників та результати розіграшу.
**Рішення:** Автоматичне збереження всіх даних в localStorage з можливістю експорту в Excel.

### 4. Нудний процес розіграшу
**Проблема:** Традиційні розіграші часто виглядають нецікаво для глядачів.
**Рішення:** Анімовані барабани, звукові ефекти та святкові popup-вікна створюють захоплюючий досвід.

### 5. Неврахування різної важливості учасників
**Проблема:** Іноді потрібно надати деяким учасникам більші шанси на виграш.
**Рішення:** Система ваг дозволяє налаштувати ймовірності для кожного учасника.

## Як це працює

### Користувацький досвід
1. **Підготовка:** Користувач додає учасників (з вагами) та призи через зручний інтерфейс або імпортує з Excel
2. **Розіграш:** Натискає кнопку "Розпочати розіграш" та спостерігає за анімованими барабанами
3. **Результат:** Отримує візуально привабливе оголошення переможця з можливістю продовжити наступний раунд
4. **Аналіз:** Може переглянути всі результати та експортувати їх для звітності

### Технічна реалізація
- **Зважений випадковий вибір:** Алгоритм враховує ваги учасників для справедливого розподілу шансів
- **Криптографічна випадковість:** Використання `crypto.getRandomValues()` замість `Math.random()`
- **Статистичні тести:** Runs Test, Chi-square Test, тест справедливості для перевірки якості алгоритму
- **Візуальні ефекти:** CSS анімації барабанів з налаштовуваними параметрами
- **Звукова система:** Base64 аудіо файли з Web Audio API fallback

- Криптографічно стійкий генератор випадкових чисел
