# Prize Raffle - Основний опис проекту

## Загальна інформація
**Назва проекту:** Prize Raffle (Розіграш призів)  
**Версія:** 2.4-sound  
**Тип проекту:** Веб-додаток для справедливих розіграшів  
**Мова інтерфейсу:** Українська  
**Мова коментарів:** Українська  

## Основна мета
Створити веб-додаток для проведення справедливих розіграшів призів з:
- Анімованими барабанами для візуального ефекту
- Зваженим випадковим вибором учасників
- Автоматичним збереженням стану
- Тестами чесності алгоритму
- Звуковими ефектами
- Excel інтеграцією

## Ключові вимоги
1. **Справедливість:** Використання криптографічно стійкого генератора випадкових чисел
2. **Зручність:** Інтуїтивний інтерфейс з автозбереженням
3. **Надійність:** Відновлення стану після перезавантаження
4. **Тестування:** Вбудовані статистичні тести для перевірки чесності
5. **Гнучкість:** Налаштовувані анімації та звукові ефекти
