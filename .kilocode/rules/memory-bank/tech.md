# Prize Raffle - Технології та налаштування

## Основні технології

### Frontend
- **HTML5** - семантична розмітка з підтримкою сучасних стандартів
- **CSS3** - стилізація з використанням Flexbox, Grid, анімацій та переходів
- **JavaScript ES6+** - сучасний JavaScript без фреймворків
- **Vanilla JavaScript** - чистий JS для максимальної продуктивності

### Бібліотеки та залежності
- **SheetJS (XLSX)** v0.18.5 - робота з Excel файлами
  - Завантажується з CDN: `https://cdnjs.cloudflare.com/ajax/libs/xlsx/0.18.5/xlsx.full.min.js`
  - Підтримка .xlsx та .xls форматів
  - Імпорт/експорт з окремих листів Excel

### Web APIs
- **localStorage API** - постійне збереження даних в браузері
- **crypto.getRandomValues()** - криптографічно стійкий генератор випадкових чисел
- **Web Audio API** - генерація синтетичних звуків (fallback)
- **File API** - робота з файлами Excel
- **Blob API** - створення файлів для експорту

## Архітектурні рішення

### Модульна система
- **Експорт через window об'єкти** - для сумісності без bundler'ів
- **Один модуль = одна відповідальність** - чіткий поділ функціональності
- **Мінімальні залежності** - кожен модуль максимально незалежний

### Система збереження
- **localStorage** як основне сховище даних
- **Автозбереження** з debounce механізмом
- **Резервне копіювання** через Excel експорт
- **Відновлення стану** після перезавантаження

## Звукова система

### Гібридний підхід
1. **Base64 аудіо дані** (пріоритет)
   - Вбудовані MP3 файли як Base64 data URLs
   - Вирішує CORS проблеми з локальними файлами
   - Самодостатність проекту

2. **Web Audio API fallback**
   - Синтетичні звуки при помилках Base64
   - Гарантована робота звукової системи

### Аудіо файли
- **drum-spin.mp3** - звук обертання барабанів (циклічний)
- **result-ding.mp3** - сигнал результату (короткий)
- **victory-fanfare.mp3** - фанфари перемоги (святковий)

## Система тестування

### Статистичні тести
1. **Runs Test** - перевірка послідовностей у генераторі
2. **Chi-square Test** - тест розподілу з урахуванням ваг
3. **Fairness Test** - комплексна оцінка справедливості

### Математичні алгоритми
- **Криптографічна випадковість** - `crypto.getRandomValues()`
- **Зважений вибір** - алгоритм з урахуванням ваг учасників
- **Статистичні розрахунки** - Z-статистика, p-value, довірчі інтервали

## Налаштування продуктивності

### Оптимізації
- **Lazy loading** - модулі завантажуються за потреби
- **Debounce** - затримка автозбереження для зменшення навантаження
- **CSS анімації** - використання GPU прискорення
- **Мінімальний DOM** - ефективні оновлення інтерфейсу

### Обмеження
- **localStorage ліміт** - зазвичай 5-10MB на домен
- **Кількість учасників** - практично необмежена (тестовано до 1000+)
- **Розмір Excel файлів** - обмежено можливостями браузера

## Безпека

### Захист даних
- **XSS захист** - використання `textContent` замість `innerHTML`
- **Валідація вводу** - перевірка всіх користувацьких даних
- **Криптографічна випадковість** - для справедливості розіграшів

### Приватність
- **Локальне збереження** - дані не передаються на сервер
- **Офлайн робота** - повна функціональність без інтернету
- **Без трекінгу** - відсутність аналітики та збору даних
