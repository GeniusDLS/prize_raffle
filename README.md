# Розіграш призів

Веб-додаток для проведення справедливих розіграшів з анімованими барабанами, зваженим вибором та статистичними тестами чесності.

[![На каву розробнику](https://img.shields.io/badge/☕_На_каву_розробнику-Monobank-black?style=for-the-badge)](https://send.monobank.ua/jar/2Rumu89c2v)

## Запуск

```bash
python -m http.server 8000
# або
npx http-server
```

Відкрийте `http://localhost:8000`. Також можна відкрити `index.html` напряму в браузері — Base64-аудіо вирішує проблему CORS на `file://`.

## Функціональність

- **Зважений вибір** — кожен учасник має числову вагу; ймовірність перемоги пропорційна вазі
- **Анімовані барабани** — два барабани (учасник / приз) з повністю налаштовуваною анімацією
- **Звукові ефекти** — обертання, сигнал результату, фанфари перемоги (Base64 MP3 + Web Audio API fallback)
- **Імпорт/Експорт Excel** — учасники та призи з окремих листів або одного листа; експорт результатів зі статистикою
- **Пресети** — 6 готових наборів тестових даних (від 5 до 1000 учасників)
- **Автозбереження** — стан відновлюється після перезавантаження сторінки
- **33 теми** — через DaisyUI, вибір збережується
- **Тести чесності** — 4 статистичні тести якості генератора та розподілу призів
- **Криптографічний RNG** — `crypto.getRandomValues()` замість `Math.random()`

## Сторінки додатку

### Дані
Дві під-вкладки — **Учасники** та **Призи**.

- Додавання / inline-редагування / видалення записів
- Сортування учасників за іменем, підрозділом або вагою (ASC/DESC)
- Перемішування учасників
- Завантаження пресету
- Імпорт з Excel / Експорт в Excel

### Розіграш
- Картки зі статистикою: кількість учасників, призів, поточний раунд
- Два барабани з анімацією
- Кнопки: **Розпочати**, **Наступний раунд**, **Новий розіграш**, **Скинути**

### Результати
- Таймлайн раундів: переможець (ім'я, посада, підрозділ) → приз
- Загальна статистика
- Експорт результатів в Excel

### Налаштування
| Під-вкладка | Що містить |
|-------------|-----------|
| Анімація | Тривалість обертання, швидкість, уповільнення, popup, звуки |
| Тести чесності | Запуск 4 статистичних тестів з вибором кількості симуляцій |
| Тема | 33 теми DaisyUI з кольоровим превью |
| Дані | Експорт / імпорт всього localStorage у JSON |

## Excel-формат

### Два окремі листи (рекомендовано)

**Лист "Учасники":**
| Ім'я | Посада | Підрозділ | Вага |
|------|--------|-----------|------|
| Іван | Розробник | ІТ відділ | 2 |
| Марія | Бухгалтер | Бухгалтерія | 1 |

**Лист "Призи":**
| Назва | Кількість |
|-------|-----------|
| iPhone 15 | 1 |
| Сертифікат | 2 |

### Один лист

Спочатку блок учасників (з заголовком або без), потім порожній рядок, потім блок призів. Колонки автоматично визначаються за позицією. Вага за замовчуванням — 1.

## Алгоритм розіграшу

1. Призи розгортаються за кількістю (2× «Сертифікат» → два окремі слоти)
2. Кожен раунд: зважений випадковий вибір учасника → випадковий приз
3. Переможець і приз вилучаються зі списків
4. Розіграш завершується, коли закінчуються призи або учасники

**Зважений вибір:** P(учасник) = вага / сума_всіх_ваг

## Тести чесності

Відкриваються в Налаштування → Тести чесності. Підмодулі завантажуються lazy при першому відкритті.

| Тест | Що перевіряє |
|------|-------------|
| **Runs Test** | Відсутність патернів у послідовності RNG (Z-статистика, p-value) |
| **Chi-square** | Відповідність частоти перемог очікуваній за вагами (χ², p-value) |
| **Тест справедливості** | Довірчі інтервали, індекс Джині, середнє відхилення (0–100 балів) |
| **Симуляція** | Монте-Карло: N віртуальних розіграшів, розподіл перемог по учасниках |

Кількість симуляцій: 100–10 000 (налаштовується для кожного тесту).

## Технології

| | |
|-|--|
| **UI** | Vanilla JS (ES6+), Tailwind CSS v4, DaisyUI v5 |
| **Excel** | SheetJS (XLSX) v0.18.5 via cdnjs |
